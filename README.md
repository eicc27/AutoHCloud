# AutoHCloud

**Screenshot function not capable on Linux for PIL.**

## Quick Setup

> Visual Studio Code with Python extenstion pack/PyCharm Community
> Python
> pip
> pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
> pip install `pyautogui`, `pyzbar`, `paperclip`, `pillow`, `pandas`
> run `main.py`
